# Question App

Приложение с различными вопросами. Вопросы можно удалять/добавлять/редактировать.

## Запуск локально

* Клонировать проект

```bash
  git clone https://github.com/yvrxslvw/Question-App
```

* Открыть solution;

* Изменить Connection String в файле App.config;

* Запустить без отладки [Ctrl + F5].
