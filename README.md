# Question App

Приложение с различными вопросами. Вопросы можно удалять/добавлять/редактировать.

## Запуск локально

* Клонировать проект

* Открыть solution;

* Создать сервисную базу данных внутри проекта Question App;

* Исполнить сначала dbo.Tests.sql код, потом dbo.Questions.sql код (ПОРЯДОК ОБЯЗАТЕЛЕН);

* Изменить значения ключа connectionString в файле App.config;

* Запустить без отладки [Ctrl + F5].
